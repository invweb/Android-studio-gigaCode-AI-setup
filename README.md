# GigaCode AI Plugin Installation Guide for Android Studio

## English

**Step 1. Add the plugin repository**  
Open Android Studio → Settings (macOS: Preferences) → Plugins. Click “Install plugin from disk…”, then select the plugin file.

**Where to get the file?**  
Download from: gitverse.ru/features/gigacode/install/ (usually a ZIP; extract and pick the JAR).

**Step 2. Restart the IDE**  
Restart Android Studio to apply changes.

**Step 3. Authorize**  
Click the GigaCode button in the bottom panel → you’ll be redirected to GitVerse. Log in with SberID. Your device will be linked to your account.

**Step 4. Check that it works**  
The GigaCode button should be visible. You can now highlight code, ask questions, etc.

## Deutsch

**Schritt 1. Fügen Sie das Plugin-Repository hinzu**  
Android Studio → Settings (macOS: Preferences) → Plugins → “Install plugin from disk…” → wählen Sie die Plugin-Datei aus.

**Wo bekomme ich die Datei?**  
Laden Sie sie von gitverse.ru/features/gigacode/install/ herunter (normalerweise ZIP; entpacken und JAR auswählen).

**Schritt 2. Starten Sie die IDE neu**  
Starten Sie Android Studio neu, damit die Änderungen wirksam werden.

**Schritt 3. Authentifizieren Sie sich**  
Klicken Sie auf die GigaCode-Schaltfläche unten → Weiterleitung zu GitVerse. Melden Sie sich mit SberID an. Ihr Gerät wird mit dem Konto verknüpft.

**Schritt 4. Überprüfen Sie, ob es funktioniert**  
Die GigaCode-Schaltfläche sollte sichtbar sein. Sie können Code markieren, Fragen stellen usw.

## 中文

**如何在 Android Studio 中安装 GigaCode AI 插件**

**步骤 1：添加插件仓库**  
打开 Android Studio → Settings（macOS 上为 Preferences）→ Plugins → 点击“Install plugin from disk…”，选择插件文件。

**在哪里获取文件？**  
从 gitverse.ru/features/gigacode/install/ 下载（通常是 ZIP，解压后选择 JAR 文件）。

**步骤 2：重启 IDE**  
重启 Android Studio 以应用更改。

**步骤 3：授权登录**  
点击底部的 GigaCode 按钮 → 跳转到 GitVerse 网站 → 使用 SberID 登录。您的设备将与账户绑定。

**步骤 4：检查是否正常工作**  
GigaCode 按钮应可见。您可以高亮代码、在插件聊天中提问等。
