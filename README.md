#Русский

Плагин GigaCode — это ИИ‑помощник внутри Android Studio: он подсказывает код, объясняет непонятное и помогает быстрее писать приложения.

# Руководство по установке плагина GigaCode AI для Android Studio

## Шаг 1. Добавьте плагин из локального файла

1. Откройте Android Studio.
2. Перейдите в настройки:
   * Windows: **File → Settings → Plugins**;
   * macOS: **Android Studio → Preferences → Plugins**.
3. Внизу страницы нажмите кнопку **Install plugin from disk…** (либо кликните на значок шестерёнки и выберите **Install Plugin from Disk…**).
4. В открывшемся окне укажите путь к файлу плагина.

### Где взять файл плагина

Скачайте архив с официального ресурса:  
`gitverse.ru/features/gigacode/install/`

Обычно загружается ZIP‑архив — распакуйте его и выберите из папки файл плагина (чаще всего это JAR‑файл).

---

## Шаг 2. Перезапустите IDE

После установки Android Studio предложит перезапустить среду разработки — сделайте это, чтобы изменения применились.

---

## Шаг 3. Авторизуйтесь в сервисе

1. После перезапуска в нижней панели IDE появится кнопка или значок GigaCode.
2. Нажмите на него — вас перенаправит на сайт GitVerse.
3. Авторизуйтесь с помощью СберID.
4. После успешной авторизации ваше устройство привяжется к аккаунту и отобразится в списке активированных устройств в личном кабинете GitVerse.

---

## Шаг 4. Проверьте работу плагина

Снова откройте Android Studio. Если установка прошла успешно:

* в интерфейсе будет видна кнопка GigaCode;
* вы сможете пользоваться помощником: выделять фрагменты кода, задавать вопросы в чате плагина, получать подсказки и т. п.

---

## Полезные советы

* Если при установке или авторизации возникает ошибка, проверьте подключение к интернету и правильность введённых учётных данных.
* В настройках плагина (через его значок) можно указать предпочтения — например, выбрать языки программирования, для которых нужны подсказки, или отрегулировать их частоту.
* При проблемах обращайтесь в поддержку: `support@gitverse.ru`.

## 中文

**如何在 Android Studio 中安装 GigaCode AI 插件**

GigaCode 插件是 Android Studio 中的 AI 助手：它会提示代码、解释难懂部分，帮你更快开发应用。

**步骤 1：添加插件仓库**  
打开 Android Studio → Settings（macOS 上为 Preferences）→ Plugins → 点击“Install plugin from disk…”，选择插件文件。

**在哪里获取文件？**  
从 gitverse.ru/features/gigacode/install/ 下载（通常是 ZIP，解压后选择 JAR 文件）。

**步骤 2：重启 IDE**  
重启 Android Studio 以应用更改。

**步骤 3：授权登录**  
点击底部的 GigaCode 按钮 → 跳转到 GitVerse 网站 → 使用 SberID 登录。您的设备将与账户绑定。

**步骤 4：检查是否正常工作**  
GigaCode 按钮应可见。您可以高亮代码、在插件聊天中提问等。
